# OpenVPN AWS Deployment

This repository contains the presentation for my **Secure OpenVPN Deployment on AWS** project.

## Overview
- Configured secure VPN on AWS EC2.
- Used PKI-based authentication for clients.
- Integrated Easy-RSA and Tunnelblick for client setup.
- Ensures private access to cloud resources inside a VPC.
